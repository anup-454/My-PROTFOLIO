🌌 Anup's Modern Portfolio Website

A modern, responsive, and animated personal portfolio website built using HTML, CSS, and JavaScript.
This project showcases projects, skills, and contact information with smooth animations and interactive UI effects.

🚀 Live Features

🎥 Animated background videos

✨ Smooth scroll animations using AOS

📱 Fully responsive design

🎬 Project video hover preview

📂 Animated sidebar menu (mobile)

🌈 Gradient animated text effects

🎠 Auto-running skills slider

📩 Contact section UI

🎨 CSS3 (Flexbox, Grid, Animations, Media Queries)
